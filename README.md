# CreditWise Loan System

## Project Overview

CreditWise Loan System is a Machine Learning project that predicts whether a loan application is likely to be approved based on applicant information and financial details.

The project uses supervised learning techniques to analyze historical loan data and assist in making loan approval decisions.

---

## Objective

The main objective of this project is to build a classification model that can predict loan approval status using applicant attributes such as:

* Gender
* Marital Status
* Education
* Employment Status
* Applicant Income
* Coapplicant Income
* Loan Amount
* Loan Term
* Credit History
* Property Area

## Machine Learning Workflow

### 1. Data Collection

Loan application dataset containing applicant and loan information.

### 3. Exploratory Data Analysis (EDA)

* Distribution analysis
* Correlation analysis
* Feature visualization

### 4. Model Building

The project uses supervised machine learning classification algorithms to predict loan approval status.

### 5. Model Evaluation

Performance is evaluated using:

* Accuracy Score
* Confusion Matrix
* Precision
* Recall
* F1 Score
